Formatação de texto

Negrito/Destaque

Maneira não semantica de utilizar o negrito com a tag <b>

Maneira semantica de utilizar o destaque com a tag <strong>

Italico / Enfase

Maneira não semantica de utilizar o italico com a tag <i>

Maneira semantica de utilizar o Enfase com a tag <em>

Texto Marcado

Maneira de se utilizar a tag<mark> que pode ter a cor modificada via CSS, nesse caso dentro do <style> na <head>

Texto grande e pequeno

estamos criando um texto grande com <big>,e um texto pequeno com <small> nesse paragrafo, cujo qual o <big> ficou obsoleto pois entra como uma tag sem semântica

Texto deletado

podemos marcar um texto como excluido com a tag <del>, que pode ser lido mas nao considerado

Texto inserido

podemos marcar um texto como inserido, com a tag <ins> para dar uma enfase e indicar que ele foi adicionado depois

Texto Sobresscrito

Para inserir coisas do tipo x20+3, com a tag <sup>

Texto Subscrito

Para inserir coisas do tipo H2O, com a tag <sub>
